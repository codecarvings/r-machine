# Contributing

Thank you for your interest in R-Machine!

This project is **not open to external contributions** at this time. Pull requests from outside collaborators will not be accepted.

## Bug Reports & Feature Requests

If you find a bug or have a feature request, feel free to [open an issue](https://github.com/codecarvings/r-machine/issues). Please include as much detail as possible to help me understand and reproduce the problem.
